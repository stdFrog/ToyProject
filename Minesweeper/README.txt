이전에 만들다 중단했던 지뢰찾기 모작입니다.

메세지 루프를 활용하면 더 빨리 반응하도록 만들 수 있습니다.
다만, 이벤트 드리븐 환경의 이해를 위해 순수 이벤트 메시지에만 반응하여 동작하도록 만들었습니다.

이미지의 경우 디테일할 필요가 없어 삭제할 부분은 삭제하고 지뢰와 깃발 비트맵만 제작했습니다.
외부 리소스없이 순수 API 코드로만 작성되어 조금 복잡해 보일 수 있으나 이벤트 순서와 트리거 등을 이해하는데에는 큰 도움이 될 것입니다.

예로, SendMessage를 호출하는 부분을 보면 작성된 순서 그대로 실행된다는 것을 알 수 있습니다.
운영체제의 동작과 역할을 확인할 수 있고, 화면의 정보를 담는 DC와 버퍼로 사용되는 Bitmap 등 여러 기법도 확인할 수 있습니다.

지뢰찾기라는 게임을 가끔 해본 적 있으나 명확한 규칙을 이해하고 있지는 않습니다.
그저 기억나는대로 구현했기 때문에 다소 부족한 부분이 있을 수 있으며, 이와 관련된 내용은 코멘트를 남겨주시거나 직접 수정하셔도 좋습니다.

작업 기간은 약 8일이 소요되었고, 시스템적으로 변화를 일으키는 함수나 동작은 일절 포함되지 않았습니다.
추후 수정할 내용이 생기면 이를 반영하여 다시 업로드 하겠습니다.
