// Version 1	- 2025.02.17

간단한 TodoList 프로그램입니다.

마지막 종료 위치를 기억하기 위해 시스템 설정(레지스트리)을 추가/변경합니다.
레지스트리 위치는 HKEY_CURRENT_USER\Software\SicSoft이며 별도의 설치 프로그램을 제공하지 않으므로 프로그램 사용/삭제시 레지스트리 값을 직접 관리해야 합니다.

또한, 사용자가 작성한 데이터를 저장하기 위해 .dat 형식의 파일을 프로그램이 위치한 경로에 생성합니다.
프로그램 종료 전 데이터를 저장할 것인지 묻기 때문에 사용자가 선택할 때만 데이터가 저장됩니다.

공개된 .cpp 파일에는 주요 로직과 함수에 대한 설명 등을 본문 코드와 함께 주석으로 작성해두었으니 참고하시기 바랍니다.
위 프로그램은 작성자의 편의에 맞춰 기능이 추가되거나 삭제될 수 있습니다.

///////////////////////////////////////////////////////////
// Version 2	- 2025.02.20
///////////////////////////////////////////////////////////
이 버전은 작업 표시줄 하단에 아이콘을 추가하고 TodaySchedule이라는 기능을 지원합니다.
메인 윈도우의 자식 윈도우를 모두 숨기고 오늘 해야할 일을 텍스트로 출력하여, 화면 오른쪽 하단에 생성합니다.
하나의 함수에 모든 기능을 작성해두었으며 필요한 때에 이 함수를 호출하여 윈도우 모드를 전환할 수 있습니다.
다만, 이 버전은 개발자가 생각없이 만든 잘못된 버전입니다.

창을 숨긴채 작업표시줄의 아이콘을 눌러 TodaySchedule 메뉴를 사용하면
화면에 잠시간 창이 생성되었다가 위치가 이동되는데 이는 로직이 잘못되었다는 의미이며,
외에 다양한 버그가 존재하기도 합니다.

///////////////////////////////////////////////////////////
// Version 3	- 2025.02.21
///////////////////////////////////////////////////////////
여러 부분에서 코드를 수정했으며, 똑같이 TodaySchedule 기능을 구현했습니다.
각 일정에 대해 개별 윈도우를 생성하여 화면 우측 하단에 위치시킵니다.
생성된 윈도우는 자유롭게 이동 가능하며 단순한 포맷의 문자열을 출력하여 보여줍니다.
이렇게 생성된 윈도우는 메인 윈도우와 완전히 분리된 상태이므로 메인 윈도우를 숨기거나 닫아도 사라지지 않으며,
우측 상단의 x 모양을 눌러야 종료됩니다.
